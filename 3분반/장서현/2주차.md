# 자바스크립트
### 데이터 타입
숫자 - 숫자 적으면 숫자로 인식  
문자열 - ""나 ''로 묶으면 문자열로 인식  
변수 - 선언시 var이나 let 붙여야 함  
상수 - 변수에 대입해야함  
주석 - //  
코드 상에서 줄바꿈 하려면 \넣으면 됨(실행 결과에는 반영 안됨)  
### 리터럴 문자열
역따옴표(`) 안에 문자열 위치시키면, \n 안하고 엔터 쳐도 실행 결과에 그냥 반영 되고, 변수도 ${변수명} 이렇게 나타낼 수 있음
### URL
localhost/**?id=HTML**  
주소에서 물음표 다음에 오는 문자열 => 쿼리 스트링  
ex) localhost:3000/?id=HTML로 접속하면 제목에 HTML 출력, localhost:3000/?id=CSS로 접속하면 제목에 CSS출력  

쿼리스트링의 문자열을 Node.js(ex. main.js) 애플리케이션이 알 수 있어야 함  
=> request.url에서 사용자가 클릭할 때마다 주소 받아옴  

require('모듈') => 모듈 요청  

response.end(queryData.id) => 쿼리 값에 따라 다른 출력
